# 第 4 步：选择代理节点

## 目标

查看可用节点，让用户选择偏好的节点。

## 操作

### 4.1 等待节点加载

首次启动后 mihomo 需要几秒钟从订阅链接拉取节点列表。等待 5 秒后继续：

```bash
sleep 5
```

### 4.2 列出可用节点

```bash
bash scripts/select-node.sh list
```

这会输出所有可用的代理节点，标有 `(active)` 的是当前使用的节点。

### 4.3 向用户展示并询问

**将节点列表展示给用户，并主动询问：**

> 以下是你的可用代理节点：
>
> （列出节点名称）
>
> 当前使用的是 **AUTO**（自动选择延迟最低的节点）。
> 你想切换到某个特定节点吗？还是继续用自动选择？

### 4.4 根据用户回复操作

- 如果用户选择了某个节点：

```bash
bash scripts/select-node.sh select "节点名称"
```

- 如果用户选择保持自动 / 不确定：无需操作，AUTO 会自动选择最快的节点。

## 完成标志

确认当前节点：

```bash
bash scripts/select-node.sh current
```

进入下一步：`skills/05-verify.md`。
